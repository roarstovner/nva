# Get a specific funding source

Retrieves information about a specific funding source by ID.

## Usage

``` r
nva_cristin_funding_source(id)
```

## Arguments

- id:

  Funding source identifier

## Value

A list with funding source details

## Examples

``` r
if (FALSE) { # \dontrun{
source <- nva_cristin_funding_source("NFR")
} # }
```
