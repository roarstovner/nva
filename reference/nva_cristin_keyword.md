# Get a Cristin keyword by ID

Retrieves a specific keyword from the Cristin registry.

## Usage

``` r
nva_cristin_keyword(id)
```

## Arguments

- id:

  Keyword identifier

## Value

A list with the keyword record

## Examples

``` r
if (FALSE) { # \dontrun{
keyword <- nva_cristin_keyword("12345")
} # }
```
