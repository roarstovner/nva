# Package index

## Search

Search publications in the NVA archive.

- [`nva_search()`](https://roarst.github.io/nva/reference/nva_search.md)
  : Search NVA publications
- [`nva_search_aggregations()`](https://roarst.github.io/nva/reference/nva_search_aggregations.md)
  : Get aggregations (facet counts) for an NVA search

## Publications

Retrieve and download publications and files.

- [`nva_publication()`](https://roarst.github.io/nva/reference/nva_publication.md)
  : Get a publication by identifier
- [`nva_publications()`](https://roarst.github.io/nva/reference/nva_publications.md)
  : Get multiple publications by identifiers
- [`nva_publication_files()`](https://roarst.github.io/nva/reference/nva_publication_files.md)
  : Get files associated with a publication
- [`nva_download_file()`](https://roarst.github.io/nva/reference/nva_download_file.md)
  : Download a file from a publication
